class View<T>

package baldrick

Represents an array of entities matched to a specific typed group of components in a Processor.

Use in a Processor as follows:

class MyProcess implements Processor {
    var myView:View<{a:ComponentA, b:ComponentB}>;

    public function process() {
        for(view in myView) {
            trace(view.entity);
            trace(view.data.a);
            trace(view.data.b);
        }
    }
}
